AI Usage Monitor

ESP32 Dashboard fuer AI Token-Verbrauch und Kosten

Dein Browser unterstuetzt Web Serial nicht. Bitte verwende Google Chrome oder Microsoft Edge auf einem Desktop-Computer.

Was ist das?

Ein kompaktes Hardware-Dashboard, das deinen AI-Token-Verbrauch in Echtzeit anzeigt. Kosten, Nutzung und Verlauf von Anthropic und OpenAI auf einen Blick.


ESP32-2432S028R (CYD 2.8")
  • Token-Verbrauch pro Provider (Anthropic, OpenAI)
  • Echtzeit-Kostentracking
  • Nutzungsverlauf und Charts
  • WiFi-Konfiguration per Captive Portal
  • OTA-Updates

Installation

  1. ESP32 per USB-Kabel an den Computer anschliessen
  2. Google Chrome oder Microsoft Edge oeffnen
  3. Unten auf "Installieren" klicken und den USB Serial Port waehlen
  4. Warten bis die Installation abgeschlossen ist

Erfordert Chrome oder Edge mit Web Serial Unterstuetzung

Troubleshooting

Kein Port sichtbar?
Stelle sicher, dass du ein USB-Datenkabel verwendest (nicht nur Ladekabel). Installiere ggf. den CP2102-Treiber oder CH340-Treiber.
Installation schlaegt fehl?
Druecke den Reset-Button auf dem Board und versuche es erneut. Stelle sicher, dass keine andere Anwendung (Arduino IDE, PlatformIO) den Port belegt.
Display bleibt schwarz nach Installation?
Trenne das USB-Kabel und schliesse es erneut an. Das Board startet automatisch neu und zeigt das WiFi-Setup an.
