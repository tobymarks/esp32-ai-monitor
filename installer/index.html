AI Usage Monitor

Standalone-Display für AI-Token-Verbrauch und Kosten

Dein Browser unterstützt Web Serial nicht. Bitte verwende Google Chrome oder Microsoft Edge auf einem Desktop-Computer.

Was ist das?

Ein kompaktes Hardware-Dashboard, das deinen AI-Token-Verbrauch in Echtzeit anzeigt. Kosten, Nutzung und Aufschlüsselung nach Modell — für Anthropic und OpenAI auf einen Blick.

ESP32-2432S028R (CYD 2.8")
  • Token-Verbrauch pro Provider (Anthropic, OpenAI)
  • Echtzeit-Kostentracking (heute und Monat)
  • Aufschlüsselung nach Modell
  • Web-basierte Konfiguration
  • Portrait- und Landscape-Modus

Installation

  1. ESP32 per USB-Datenkabel an den Computer anschließen
  2. Google Chrome oder Microsoft Edge öffnen
  3. Unten auf „Firmware installieren“ klicken und den USB-Port wählen
  4. Warten bis die Installation abgeschlossen ist (~30 Sek.)
Nach der Installation: Das Display zeigt ein WiFi-Setup. Verbinde dich mit dem Netzwerk AI-Monitor-Setup und konfiguriere dein WLAN. Danach erreichst du die Konfiguration unter http://ai-monitor.local

Erfordert Chrome oder Edge mit Web Serial

Troubleshooting

Kein Port sichtbar?
Stelle sicher, dass du ein USB-Datenkabel verwendest (nicht nur Ladekabel). Installiere ggf. den CP2102-Treiber oder CH340-Treiber.
Installation schlägt fehl?
Drücke den Reset-Button auf dem Board und versuche es erneut. Stelle sicher, dass keine andere Anwendung (Arduino IDE, PlatformIO) den Port belegt.
Display bleibt schwarz?
Trenne das USB-Kabel und schließe es erneut an. Das Board startet automatisch neu und zeigt das WiFi-Setup an.
