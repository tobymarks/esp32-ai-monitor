AI Monitor

Dein Claude-Nutzungslimits immer im Blick

Dein Browser unterstützt Web Serial nicht. Bitte verwende Google Chrome oder Microsoft Edge auf einem Desktop-Computer.

Was ist das?

Ein kompaktes Desk-Display, das deine Claude Pro/Max Nutzungslimits in Echtzeit anzeigt. Session-Limits (5h), Wochenlimits und Kosten-Tracking — per USB mit deinem Mac verbunden, kein WiFi nötig.

  • Session- und Wochenlimits mit Fortschrittsbalken
  • Live Reset-Countdown
  • Kosten-Tracking ($used / $limit)
  • 1-Minuten-Polling, immer aktuell
  • Automatische USB-Erkennung
  • Portrait- und Landscape-Modus

Was du brauchst

  1. ESP32-2432S028R Board (~8 $, „Cheap Yellow Display“)
  2. USB-Datenkabel (kein reines Ladekabel)
  3. Mac mit macOS 13+ (Apple Silicon)
  4. Claude Pro oder Max Abo
  5. Claude Code CLI installiert

Setup in 3 Schritten

  1. Firmware unten flashen (oder via PlatformIO)
  2. AI Monitor Mac App herunterladen (via GitHub Releases)
  3. ESP32 per USB an den Mac anschließen — fertig!
So funktioniert es: Die Mac App liest deine Claude-Limits über die CLI aus und sendet die Daten per USB-Serial an das Display. Kein WiFi, kein Cloud-Dienst — alles lokal.

Firmware flashen

Erfordert Chrome oder Edge mit Web Serial

Troubleshooting

Kein Port sichtbar?
Stelle sicher, dass du ein USB-Datenkabel verwendest (nicht nur Ladekabel). Installiere ggf. den CP2102-Treiber oder CH340-Treiber.
Flash schlägt fehl?
Drücke den Reset-Button auf dem Board und versuche es erneut. Stelle sicher, dass keine andere Anwendung (Arduino IDE, PlatformIO, AI Monitor App) den Port belegt.
Display bleibt schwarz nach dem Flashen?
Trenne das USB-Kabel und schließe es erneut an. Das Board startet automatisch neu.
Mac App erkennt das Display nicht?
Prüfe, ob der USB-Treiber (CP2102 oder CH340) installiert ist. Unter ls /dev/tty.usb* sollte ein Gerät erscheinen.
